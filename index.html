About HornetMM

HornetMM is a community-driven mod manager for Hollow Knight, built for simplicity and performance. Manage your mods, install updates, and explore new creations—all in one place.

Key Features

  • 🧩 One-click mod installs
  • ⚙️ Auto-updates for installed mods
  • 🌐 Cross-platform support
  • 💾 Easy backups & restores
